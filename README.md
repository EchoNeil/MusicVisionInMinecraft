# MusicVisionInMinecraft
一个用c++编写的在minecraft中创建数据包实现音乐可视化的工具  
**效果**:[https://www.bilibili.com/video/BV1MSZTBLEgL/](https://www.bilibili.com/video/BV1MSZTBLEgL/)


## 如何使用
- **版本**：支持1.20+
- 下载MusicVision_minecraft.exe后在文件所在目录打开cmd并输入:
- MusicVision_minecraft.exe <音频文件路径> <输出数据包文件路径> <起始点x> <起始点y> <起始点z>
- 将对应数据包加载到minecraft中即可

## 使用的库
- **kiss_fft**：[https://github.com/mborgerding/kissfft](https://github.com/mborgerding/kissfft)
- **minimp3**：[https://github.com/lieff/minimp3](https://github.com/lieff/minimp3)
